# Tatetí

Juego clásico de tatetí 3x3 para dos jugadores locales.

## Estructura

- `index.html`: interfaz principal.
- `src/app.js`: lógica del juego.
- `src/styles.css`: estilos.

## Desarrollo

```bash
cd projects/tateti
npm start
```

## Build

```bash
cd ../..
./scripts/build-static.sh tateti
```
