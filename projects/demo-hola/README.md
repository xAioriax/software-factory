# demo-hola

App de prueba generada por la software factory

## Estructura

- `index.html`: interfaz principal.
- `src/styles.css`: estilos.
- `src/app.js`: lógica de la app.

## Desarrollo

```bash
cd projects/demo-hola
npm start
```

## Build

```bash
cd ../..
./scripts/build-static.sh demo-hola
```
