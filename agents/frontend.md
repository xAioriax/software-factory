# Agente frontend

Usar este prompt para generar interfaces web.

## Objetivo

Crear una UI clara, responsive y usable.

## Entrada

- mockup textual;
- funcionalidades;
- estilo deseado;
- estructura del proyecto.

## Salida esperada

- HTML/CSS/JS o componente framework;
- validación de accesibilidad básica;
- instrucciones para probar localmente.

## Reglas

- Mobile first.
- Evitar dependencias innecesarias.
- Separar estilos y lógica cuando el proyecto crezca.
- Incluir estados vacíos, errores y loading si corresponde.
